CMS Style Guide

Overview

Currently, there is website referred to as the "web style guide" or "style guide" that showcases the UI patterns available within the CMS for use on the marketing website.

The CMS Development Team (PO and developers) and Web Production Team use the web style guide for the following:

  • referencing CMS component variants, testing (regression, general QA)
  • referencing utility classes for text, color, typography, and spacing
  • referencing HTML markup for custom uses or custom pages
