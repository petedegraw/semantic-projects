Posts by Language

  1. Select the language from the country dropdown menu

  2. View the link header in the network tab ![Blog Feed Country API Endpoints](Blog Feed Country API Endpoints.png)

  3. Get the country ID 4. Use the Posts endpoint with country argument to get in language posts

    e.g. https://blog.equinix.com/wp-json/wp/v2/posts/?country=2451&per_page=1
