Wordpress Installation

In order to install Wordpress, please follow detailed instructions in Wordpress official codex.

Theme Installation

To install the theme, you should already have a working version of WordPress.

First, extract the zipped package downloaded from ThemeForest to your chosen folder. In the extracted package, you should find appic.zip which is the WordPress theme file.

So, there are two ways of theme installation - via Wordpress Admin panel and via FTP.

Admin Panel

In Wordpress Admin panel, navigate to Appearance > Themes in the left sidebar. On the top of the page there will be a link Add New, click on that link and after you get on Install Themes page, click Upload link. Select appic.zip file and press Install Now button. As soon as your theme is installed, you should see an Activate link. Click on it. Hurrah! Your Appic theme is now successfully activated.

FTP

Using FTP, extract your appic.zip file and upload the extracted folder to /wp-content/themes/ folder on your server. Now you have to activate the theme. To do that, go to Wordpress Admin panel and navigate to Appearance > Themes in the left sidebar. On the loaded page, find your Appic theme image, hover over it and press Activate button. The theme is now activated.

Make your Theme Look like our Live Demo

To make your Appic theme look like our live demo, please follow instructions described in the next steps. Please NOTE, images from our live demo will not be imported. They are under a license and therefore are replaced with placeholders. You can replace those in the respective locations. For example, you would to update a post image. To do this, open this post in the Editor and in the right sidebar find Featured Image link to upload a different picture.

To complete the setting up, go to the next section Setting Up Revolution Slider.

Shortcodes

Appic Wordpress theme provides you with over 50 custom shortcodes. Additionaly, you may want to use Wordpress built-in shortcodes. To insert a shortcode, you should first create a page. To do this, go to Pages tab in the left sidebar in Admin panel and in the dropdown menu select Add New. In the top bar, just above the text area box, there is a Shortcodes icon. Click on it and select your desired shortcode.

You may find it difficult to use some of these shortcodes. Therefore, the most complex are described below.

Grid

The Appic layout is built on Bootstrap 2.3 grid system. This means that the content area is divided on 12 columns, making total width of 940px. So, first you should select a Row shortcode (Grid > Row) and inside row tags [row][/row] insert a column shortcode (Grid > Column ...). For example, you want the content to be divided into three columns (3/12 of the content width, then 6/12 and then again 3/12). The code representing this content layout is shown below:

[row][column_3][/column_3][column_6][/column_6][column_3][/column_3][/row]

You should remember that the sum of numbers in the end of columns should always equal 12. Alternatively, you may want to use offsett Bootstrap classes, so your code can be modified as follows:

[row][column_3][/column_3][column_3 class="offset6"][/column_3][/row]

To the sum of column numbers, add the number in the end of offset class. This sum should also equal 12. For detailed information on Grid system and Bootstrap classes, please refer to Bootstrap official documentation.

Pricing Tables

You may find Pricing Tables shortcode a little complicated. When you insert this shortcode (Shortcodes > Elements > Pricing tables > 4 columns), the code pasted into your content area is as following:

[pricing_table columns="4"][pricing_column title="Title" price="$ 0.00" button_text="Purchase" button_url="#"][pricing_item]Parameter 1[/pricing_item][pricing_item]Parameter 2[/pricing_item][/pricing_column][pricing_column title="Title" title_class="" price="$ 0.00" button_text="Purchase" button_url="#" button_class=""][pricing_item]Parameter 1[/pricing_item][pricing_item]Parameter 2[/pricing_item][/pricing_column][/pricing_table]

To clarify a little bit: [pricing_table][/pricing_table] is the container for pricing columns. Pricing columns are marked as [pricing_column][/pricing_column] and serve as container for rows of information marked as [pricing_item][pricing_item]. By default, two pricing columns are inserted, one standard and one premium. Depending on how many columns you choose, you can append before the ending tag [/pricing_table] one, two or three additional pricing columns.

Table

Shortcode Table is located at Shortcodes > Elements > Table. The code inserted into the content area is as following:

[table][head][table_row][column]Title 1[/column][column]Title 2[/column][column]Title 3[/column][column]Title 4[/column][column]Title 5[/column][/table_row][/head][body][table_row][column]1[/column][column]2[/column][column]3[/column][column]4[/column][column]5[/column][/table_row][table_row][column]1[/column][column]2[/column][column]3[/column][column]4[/column][column]5[/column][/table_row][/body][/table]

This shortcode will draw a table of three rows and five columns. If you would like to append an additional column, then add [column][/column] before all closing tags [/table_row]. If you would like to add an additional row, just add [table_row]([column][/column] * your number of columns)[/table_row] before the closing tag [/body].

Recommended Plugins

The Appic theme comes with five recommended plugins. Below there is a short review for each of them.

Revolution Slider

With Appic theme comes the most popular Wordpress slider plugin - Revolution Slider (you have saved $19!). For detailed documentation for the plugin use, see here. Two styles of the Home page are coming with two ready-to-use Revolution Sliders, which you can import from our live demo (see above). Also, you can embed Revolution Slider on any of your page through its shortcode.

Contact Form 7

With Contact Form 7, you can create as many and as complex forms as you like. For more detailed plugin documentation, see here. The Contact Form 7 is integrated with Appic styling, you can see the example of the form on the Contact page. To reproduce the Appic styling in any other forms that you create, add to the input fields and textareas class input-style. A different styling of Contact Form 7 is applied on Appic: Subscribe to News widget which in live demo is located in the widget area named Footer 1 & 2. One more form style can be observed on the Coming Soon page.

Mailchimp

Mailchimp Wordpress plugin is built for Mailchimp subscription service. This service allows you gathering subscribers to the lists and then sending email marketing campaigns. For the plugin documentation, see here. The Appic styling is applied in Mailchimp plugin on Appic: Subscribe to News widget located in the live demo in the widget area named Footer 1 & 2 and on the Coming Soon page. By default, these two locations are set up with Contact Form 7 forms, but you can easily change them to Mailchimp forms. First, you should set up the Mailchimp plugin. After that, in the respective locations replace Contact Form 7 shortcode with Mailchimp shortcode [mailchimpsf_form].

OAuth Twitter Sidebar Widget

OAuth Twitter Sidebar Widget is applied in the third footer widget area. To set up this widget, please go the above section called Setting Up Widgets.

WPML

By having purchased Appic Wordpress theme, you have wisely saved $79. WPML or The WordPress Multilingual Plugin is the most used plugin for Wordpress website translation. It offers rich features, for detailed documentation see here.

Theme Translation

Appic theme comes with the translation plugin named WPML. You can use the plugin to translate your theme in as many languages as you like. Alternatively, you can follow steps desribed below.

Let's say you would like your theme to be running in French. First, download and install the program called POEdit. Open the program and in the opened window click Create new translation and choose the translation file located in /wp-content/themes/appic/languages/en_En.po of your Wordpress installation and from the dropdown menu select French. Now you can make the translations. When you are finished, press Save button on the top toolbar and save it to the same location /wp-content/themes/appic/languages/fr.po.

After, open wp-config.php located in the root folder of your Wordpress and search for define('WPLANG', 'en_CA'); string, now change it to: define('WPLANG', 'fr');. You should notice that fr is the name of your translated file.

Updating Theme

To be notified of new Appic updates, you should turn on Update Notifier button in Appearance > Theme Options > General (is enabled by default):

There are two locations on the website where you can see that there is a new version of Appic theme available:

  • on the top Admin toolbar
  • in Admin panel > Appearances > Appic

To update your Appic theme, you should first delete its current version. To do this, in Admin panel go to Appearance > Themes and activate another theme. Then click on Appic theme thumbnail and in the popup window in the bottom right corner click Delete link.

Now go to Theme Installation section of this documentation to install and activate the new version of Appic theme. After that, in Admin panel go to Appearance > Theme Options and click Save Changes button to save the new options.
